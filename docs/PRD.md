# PRD: 데일리 워크로그 & 스크럼 자동 생성기

> 버전: 1.0
> 작성일: 2026-02-22
> 상태: 초안

---

## 1. 문제 정의

매일 반복되는 데일리 스크럼 작성이 번거롭고, 전날 작업 내용을 정리해서 팀에 공유하는 과정에 불필요한 시간이 소요된다.

### 해결하려는 고통

- 퇴근 전/출근 후 스크럼 작성에 매일 10~20분 소모
- 전날 뭘 했는지 기억해서 정리하는 과정이 번거로움
- 팀마다 다른 공유 포맷(슬랙, 노션 등)에 맞게 변환하는 반복 작업

---

## 2. 목표

| 목표 | 측정 기준 |
|------|-----------|
| 작업 기록 시간 단축 | 하루 5분 이내로 로그 입력 완료 |
| 스크럼 자동 생성 | 버튼 1번으로 공유용 문서 완성 |
| 팀 공유 간소화 | 복사 또는 링크 공유 즉시 가능 |

---

## 3. 사용자

- **주 사용자:** 팀에서 데일리 스크럼을 공유하는 개발자 (1인)
- **공유 대상:** 팀원 (읽기 전용, 계정 불필요)

---

## 4. 핵심 사용자 플로우

```
[퇴근 전]
오늘 한 일 입력 (작업 항목 + 상태 태그)
        ↓
[AI 정리]
작업 내용 자동 요약 + 포맷팅
        ↓
[다음날 아침]
"스크럼 생성" 버튼 클릭
        ↓
[자동 완성]
어제 한 일 + 오늘 할 일 초안 생성 → 수동 수정 가능
        ↓
[공유]
슬랙/노션에 붙여넣기 or 링크 공유
```

---

## 5. 기능 명세

### 5-1. 워크로그 입력

- 날짜별 작업 항목 추가 / 수정 / 삭제
- 입력 형태: 자유 텍스트 (마크다운 지원)
- 작업 태그: `기능개발` / `버그픽스` / `회의` / `리뷰` / `기타`
- 진행 상태: `완료` / `진행중` / `블로킹`
- 오늘 날짜 자동 선택, 날짜 변경 가능

### 5-2. AI 스크럼 생성

- 전날 워크로그 → **어제 한 일** 자동 요약
- `진행중` / `블로킹` 항목 → **오늘 할 일** 초안 자동 생성
- 생성 결과 수동 수정 가능
- 출력 포맷 2가지 선택:
  - **슬랙 포맷** - 이모지 + 블릿 형식
  - **마크다운 포맷** - 노션/문서 호환

### 5-3. 공유

- 원클릭 클립보드 복사
- 고유 공유 링크 생성 (읽기 전용, 만료 없음)
- 공유 페이지는 로그인 불필요

type WorkLog = {
  id: string

  items: WorkItem[]
  createdAt: Date

  id: string
  content: string     // 작업 내용 (자유 텍스트)

## 8. 페이지 구성

| 경로 | 설명 |
|------|------|
| `/` | 오늘의 워크로그 입력 (메인) |
| `/scrum` | 스크럼 생성 & 미리보기 & 공유 |
| `/history` | 날짜별 워크로그 기록 조회 |
| `/share/[id]` | 공유 링크 (읽기 전용, 로그인 불필요) |
